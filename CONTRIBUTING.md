# 🤝 Contributing to FoodShare AI

Thank you for your interest in contributing to FoodShare AI 🚀

We welcome contributions from developers of all skill levels.

---

# 📌 Project Overview

FoodShare AI is an AI-powered food waste management platform that connects food donors with nearby NGOs in real-time.

---

# 🛠️ Tech Stack

## Frontend
- Next.js
- TypeScript
- Tailwind CSS
- ShadCN UI

## Backend
- Node.js
- Express.js
- MongoDB

---

# 🚀 Getting Started

## 1️⃣ Fork the Repository

Click the Fork button at the top right of the repository.

---

## 2️⃣ Clone Your Fork

```bash
git clone https://github.com/your-username/FoodShareAI.git
```

---

## 3️⃣ Install Dependencies

### Frontend

```bash
cd Frontend
npm install
npm run dev
```

### Backend

```bash
cd Backend
npm install
npm run dev
```

---

# 🌿 Create a Branch

```bash
git checkout -b feature/your-feature-name
```

Example:

```bash
git checkout -b fix/navbar-responsive
```

---

# ✅ Contribution Guidelines

- Write clean and readable code
- Follow existing project structure
- Test your changes before submitting
- Avoid unnecessary files
- Keep pull requests focused on one issue

---

# 📝 Commit Message Format

```bash
git commit -m "fix: improve navbar responsiveness"
```

---

# 🚀 Submitting a Pull Request

1. Push your changes
2. Open Pull Request
3. Mention the related issue number

Example:

```txt
Closes #12
```

---

# ⚠️ Important Notes

- Ask to be assigned before working on an issue
- PRs without issue references may not be accepted
- Inactive issues may be reassigned after 3 days

---

# 🌟 Need Help?

Feel free to open a discussion or ask questions in issue comments.

Happy Coding 🚀
